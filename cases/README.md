# 오박사만능인테리어 HTML → Markdown 변환본

기존 `cases/2026/*.html` 정적 게시물을 Markdown 원본 형태로 변환한 파일입니다.

- 원본 HTML 파일은 수정하지 않았습니다.
- 이미지 경로는 기존 HTML의 `src` 값을 그대로 유지했습니다.
- SEO 메타정보는 YAML front matter로 최대한 보존했습니다.
- 본문은 H1/H2/H3, 문단, 이미지, 목록 중심으로 변환했습니다.

## 폴더 구조

```text
posts_md/2026/*.md
```

## 사용 방식

1. 앞으로는 Markdown을 원본 글로 관리합니다.
2. Python 빌드 스크립트가 Markdown을 읽어 기존 HTML 템플릿에 넣어 `cases/2026/*.html`을 생성합니다.
3. 기존 이미지 폴더와 URL 구조는 그대로 유지합니다.

## 주의

현재 파일은 1차 역변환본입니다.
실제 운영용 자동 빌드에 넣기 전에는 중복 페이지 canonical 정리와 FAQ/CTA 템플릿 자동화를 함께 적용하는 것이 좋습니다.
